Chrome

Google Chrome is a freeware web browser[12] developed by Google. It was first released in 2008 only for Microsoft Windows, but was eventually ported to Linux, OS X and iOS and Android.

FireFox

Mozilla Firefox (or simply Firefox) is a free and open-source[21] web browser developed by the Mozilla Foundation and its subsidiary, the Mozilla Corporation.

IE

Internet Explorer[a] is a discontinued series of graphical web browsers developed by Microsoft and included as part of the Microsoft Windows line of operating systems, starting in 1995.

Opera

Opera is a web browser developed by Opera Software. The latest version is available for Microsoft Windows, OS X, and Linux operating systems, and uses the Blink layout engine.

Safari

Safari is a web browser developed by Apple based on the WebKit engine. First released in 2003 with Mac OS X Panther, a mobile version has been included in iOS devices since the introduction of the iPhone in 2007.
